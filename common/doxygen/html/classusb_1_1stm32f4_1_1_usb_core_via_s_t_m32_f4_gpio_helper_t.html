Detailed Description

template<gpio::GpioAccessViaSTM32F4::Function_e GpioFunctionT>
class usb::stm32f4::UsbCoreViaSTM32F4GpioHelperT< GpioFunctionT >

Helper Class / Type to encapsulate the GPIO Configuration.

GPIO Configuration of the USB Pins depends on the used USB Function (High vs. Full Speed).

This helper class encapsulates this in a type-safe way.
