"Get Status" Response for USB 2.0 Devices

This structure defines the Response to a Get Status Control Command Response for USB Devices as per Figure #9-4 "Information Returned by a GetStatus() Request to a Device" of the "Universal Serial Bus Specification" Revision 2.0 from April 27, 2000
